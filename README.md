
# 🍕 Italian Pizza - Premium Ordering Experience


> **"Where Luxury meets Taste."** > A high-end web application designed with a **Dark Academia** aesthetic, featuring dynamic menu management, real-time cart logic, and premium CSS glassmorphism.

---

## 🌟 Key Highlights

### 🎨 Visual Excellence
- **Dark Luxury UI:** Focused on a "Moody" aesthetic using deep charcoal backgrounds and **Gold (#D4AF37)** accents.
- **Glassmorphism:** Modern blurred-background cards for a premium "Glassy" feel.
- **Micro-interactions:** Smooth hover effects and button animations for a polished user experience.

### ⚙️ Core Functionality
- **Dynamic Search:** Instant "Live Filter" to find pizzas from the menu without page reloads.
- **Customization Modal:** Real-time size selection and topping additions with automatic price calculation.
- **Advanced Cart Logic:** - Persistent storage using `localStorage`.
  - Automated Coupon System (`WEEKEND50`, `BOGO`, `FIRST100`).
  - Dynamic Tax & Delivery fee breakdown.
- **Fully Responsive:** Seamlessly works on Mobile, Tablet, and Desktop using **Bootstrap 5**.
